Q. Consider the following  code snippet  for jQuery selector:

$("div#intro .head"). 
What does it select? 

1). All elements with class="head" inside the first div element 
with id="intro" 
2). The first element with id="head" inside any div element with class="intro" 
3). All div elements with id="intro" and all descendants of div
 with class="head" 
4). First element with id=“intro” or class=“head” 
Solution :
option [3] is correct
